**# BindingPDF**
Утилита предназначена для упрощения работы с пакетами отгрузочных документов.
Она сканирует директории, находит нужные PDF-файлы по ключевым словам и объединяет их в единый файл согласно выбранному сценарию.
**Ключевые возможности:**
Гибкая выборка: 
Обработка папок по заданному диапазону номеров инвойсов (например, 3550-3560, 3575).
Сценарии скрепления:
- Инвойсы и Спецификации: Собирает файлы Invoice*.pdf и сортирует их.
- Декларации и ЭСД: Находит GTD_*.pdf и соответствующие ЭСД файлы.
- Пакет: ГТД + Инвойс + Спецификация (сортировка по номерам ГТД).
- Только ГТД: Выборка одних таможенных деклараций.
- Ручной режим (Temp): Быстрое скрепление произвольных файлов из папки Temp, отсортированных по префиксу в имени (напр. 1,Doc.pdf, 2,Doc.pdf).
Автоматическое наименование скрепленного документа:
Выходной файл автоматически получает имя, содержащее диапазон обработанных номеров и количество документов.
