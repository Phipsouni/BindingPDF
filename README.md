**# BindingPDF**

Утилита предназначена для упрощения работы с пакетами отгрузочных документов
Сканирует директории, находит нужные PDF-файлы по ключевым словам в названии файлов и объединяет их в единый файл согласно выбранному сценарию


- Обработка папок по заданному диапазону номеров инвойсов (например, 3550-3560, 3575)
Сценарии скрепления:
- Инвойсы и Спецификации: скрепление файлов Invoice*.pdf по порядку возрастания номера инвойса
- Декларации и ЭСД: скрепление GTD_*.pdf и соответствующих ЭСД по порядку возрастания номера инвойса
- Пакет: ГТД + Инвойс + Спецификация: скрепление по возрастанию номера GTD
- Только ГТД: Скрепление только таможенных деклараций
Выходной файл автоматически получает имя, содержащее диапазон обработанных номеров и количество документов
- Ручной режим: скрепление произвольных файлов из папки Temp (должна быть в корневой папке скрипта), скрепление в порядке числа перед запятой в названии файла (напр. 1,Doc.pdf, 2,Doc.pdf)
- Ж/Д накладные: скрепление ЖДн по 4 шт. в порядке возрастания числа в названии файла
